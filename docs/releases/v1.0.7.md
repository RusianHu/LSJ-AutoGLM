# LSJ-AutoGLM v1.0.7

## 变更摘要

- 正式切换为 Windows x64 目录分发方案，不再构建单文件 exe。
- 新增独立的干净发布目录，避免日常 `dist` 中的 `.env`、历史记录或日志进入发布包。
- 内置经过白名单精简的 Android Platform-Tools、scrcpy 与 ADBKeyboard.apk。
- 发布包附带 `.env.example`、`LICENSE`、`README.md` 和 `BUILD-INFO.txt`。
- 构建后自动执行冻结任务子进程、adb 和 scrcpy 冒烟测试，并生成 ZIP 及 SHA-256 校验文件。
- 新增发布包解压验证脚本，检查 GUI 启动、首次 `.env` 生成、敏感默认值、任务子进程和内置运行时。

## 分发说明

1. 下载 `OpenAutoGLM-GUI-v1.0.7-windows-x64.zip`。
2. 校验同名 `.sha256` 文件后完整解压 ZIP。
3. 运行目录中的 `OpenAutoGLM-GUI.exe`。
4. 首次启动会自动生成 `.env`，API Key 需由用户在设置页填写。

不要只复制 exe；`_internal` 目录是必需的运行时内容。
