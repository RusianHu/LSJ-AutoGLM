# LSJ-AutoGLM v1.0.11

## 变更摘要

- 修复 MiMo 思考模式多轮调用协议：独立采集模型返回的 `reasoning_content` 与 `content`，并在后续每次请求中原样回传全部历史 `reasoning_content`。
- 保留非推理模型和旧式 `<think>/<answer>` 输出的兼容路径；仅当上游实际返回推理字段时才向历史消息添加 `reasoning_content`。
- 修复 Android `Find_App` 中文名称搜索：复用内置应用名到包名映射，并在快速包扫描结果中附加已知中文显示名。
- 新增重复页面转移保护：同一画面上的相同坐标动作连续两次进入同一页面后，第三次会在执行前被拦截并要求模型更换策略。
- 新任务启动时清理页面转移记录，避免不同任务之间相互影响。
- 新增 MiMo 流式字段分离、多轮历史回放、中文查包和循环拦截回归测试。

## 分发说明

1. 下载 `OpenAutoGLM-GUI-v1.0.11-windows-x64.zip`。
2. 使用同名 `.sha256` 文件校验完整性后，完整解压 ZIP。
3. 运行目录中的 `OpenAutoGLM-GUI.exe`。
4. 首次启动会自动生成 `.env`，API Key 需由用户在设置页填写。

不要只复制 exe；`_internal` 目录是必需的运行时内容。
