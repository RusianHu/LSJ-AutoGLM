# LSJ-AutoGLM v1.0.8

## 变更摘要

- 所有模型渠道统一使用标准 Phone Agent 提示词，移除第三方提示词模式及相关环境变量、CLI 开关和 GUI 标签。
- 截图压缩改为独立运行选项，默认关闭以保留识别质量；TUI、GUI 与非阻塞 CLI 使用相同配置语义。
- 重构 Android 11+ 二维码配对：精确匹配本次 `studio-*` mDNS 配对服务，并区分临时配对端口与 TLS 连接端口。
- 修复 ADB 35+ 将真实配对错误包装为 `protocol fault` 时的误诊：从 `adb server-status` 指向的日志还原 TCP 超时、端口关闭和地址错误。
- 二维码配对首次失败后自动刷新 ADB mDNS 状态；Windows 目标邻居被标记为 `Unreachable` 时，只清理该手机的失败 ARP 项并重试。
- 优化二维码对话框状态说明，避免长错误覆盖二维码；完整诊断保留在操作日志和悬浮提示中，中英文 i18n 已同步。
- 增强无界面设备诊断，`device adb-status` 现在同时输出 ADB server 版本、mDNS 后端、日志和密钥路径信息。
- 同步官方上游 `README.md` 到 `README_official.md`。

## 分发说明

1. 下载 `OpenAutoGLM-GUI-v1.0.8-windows-x64.zip`。
2. 使用同名 `.sha256` 文件校验完整性后，完整解压 ZIP。
3. 运行目录中的 `OpenAutoGLM-GUI.exe`。
4. 首次启动会自动生成 `.env`，API Key 需由用户在设置页填写。

不要只复制 exe；`_internal` 目录是必需的运行时内容。
