# LSJ-AutoGLM v1.0.9

## 变更摘要

- GUI 主界面切换为横版侧边栏布局：左侧固定宽度侧边栏（品牌 / 纵向导航 / 主题切换），右侧内容区；工作台改为 QSplitter 左右分栏，左列为控制区与日志/概览 Tab，右列为常驻设备镜像，不再与日志抢占同一块空间。
- 主题系统升级至 v3：暗色 "Obsidian"（近黑中性底 + 侧边栏纵深）与浅色 "Porcelain"（瓷白冷灰底），accent 统一为靛紫；重排任务/镜像状态色与语义色饱和度以保证可读性。
- 引入 Font Awesome 4 图标字体系统，侧边栏导航按钮重设计为"图标 + 文字 + 选中指示条"结构，颜色状态由 Python 驱动子标签样式以规避 QSS 祖先伪态选择器的兼容性问题。
- 新增布局工具 `wrap_center_column`（居中限宽列，避免表单/卡片在宽窗口下无限拉伸），新增离屏视觉验证脚本 `scripts/gui_screenshot.py`（不弹窗输出到 `build/ui_preview/`）。
- 全局背景基色改用 QPalette 而非全局 QWidget QSS，避免祖先样式表级联给 QLabel 涂底色块；壳层与栈容器 QSS 统一使用作用域选择器，图标字体族写进控件自身样式表以防被全局 font-family 覆盖。
- 各页面与样式文件（banners/buttons/cards/dialogs/inputs/lists/logs/navigation/shell）适配新主题与配色。

## 分发说明

1. 下载 `OpenAutoGLM-GUI-v1.0.9-windows-x64.zip`。
2. 使用同名 `.sha256` 文件校验完整性后，完整解压 ZIP。
3. 运行目录中的 `OpenAutoGLM-GUI.exe`。
4. 首次启动会自动生成 `.env`，API Key 需由用户在设置页填写。

不要只复制 exe；`_internal` 目录是必需的运行时内容。
